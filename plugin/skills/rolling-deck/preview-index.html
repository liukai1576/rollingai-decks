快速上手

  • 新增一页：在 template.html 里复制任意 <section class="slide">，页码、进度条自动更新
  • 改文字：演示中点右下角 ⚙ → 编辑模式，点击页面文字直接改，自动保存
  • 加图标：<i data-lucide="rocket"></i>，彩色方块用 .ico-chip blue|violet|green|fire
  • 加动效：data-anim="rise|left|right|scale|blur"、data-stagger、data-count="98%"
  • 直达某页：template.html?slide=22；导出 PDF：⚙ 面板内一键导出
